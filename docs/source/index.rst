ValBal-Trajectory documentation
===============================

.. note::
   本文档是 ValBal-Trajectory 的介绍性文档，用于内部讨论和关键模块的详细阐述

ValBal-Trajectory 是一个用于实现高空气球轨迹规划的项目，该项目适用于通过改变气球在三维空间中的竖直高度，进而实现路径规划的相关任务。
本项目主要包含四个部分：1.大气信息的获取与处理 2.高度调节机制与控制策略 3.定点通过路径规划与优化 4.气球轨迹仿真与硬件在环测试。
